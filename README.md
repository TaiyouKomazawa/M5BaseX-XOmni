# M5BaseX-XOmni



## Overview

　M5Stack及びBaseXを用いてev3Mモータを制御して４輪のオムニホイールを制御する。

## Environment
 
　以下環境にて動作確認。

 ### Arduino IDE
 
 	- M5Stack 1.0.7 (Core)
 	- M5Stack 0.3.1 (Library)
 
 ###　Hardware
 
 	- M5Stack Gray
 	- M5StackModule BaseX
 	- LEGO EV3 M-Motor x4
 	- Φ48mm Omni wheel for LEGO (Nexus robot) x4
